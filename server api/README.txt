ראשית להיכנס לתיקייה json-server-master ולהריץ את הפקודה בCMD כADMIN
npm install -g json-server

לצאת חזרה לתיקיית server api ולהריץ את הפקודה
npm install json-server --save-dev

לאחר מכן להריץ את הפקודה
json-server --watch users.json

לאחר מכן לפתוח CMD אחר כADMIN ולהריץ את הפקודה
node server
